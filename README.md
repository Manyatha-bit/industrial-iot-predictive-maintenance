
# Industrial IoT Predictive Maintenance System

## Overview
An ESP32-based Industrial IoT system for predictive maintenance using:
- LM35 Temperature Sensor
- ACS Current Sensor
- MPU6050 Vibration Sensor
- L298N Motor Driver
- ThingSpeak Cloud Platform

## Features
- Real-time monitoring
- Fault detection
- Cloud analytics
- LED alert system
- Automatic motor shutdown

## Technologies Used
- ESP32
- Arduino IDE
- ThingSpeak
- Embedded C
- IoT

## Output
- Serial monitor visualization
- ThingSpeak dashboard
- Predictive fault indication
